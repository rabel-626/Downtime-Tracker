Event Log

CSV Ready
The Event Log is the main downtime record. Timer mode captures timestamps automatically; Manual Entry mode simplifies the sheet to event duration, flag/reason code, and notes.
  • Add Manual Row lets you enter or correct an event manually.
  • Copy Selected Columns copies only the checked columns below into an Excel-ready paste format.
  • Copy Column Selection controls what gets copied; it does not hide or delete data from the log.
  • Downtime Code options come from the Reason Code Setup section below the event log.
  • The Start/Stop Downtime bars above and below the table mirror the main capture button and span the full table width for faster logging while entering notes.
  • Downtime cannot be started unless the study timer is running. Press Start Study first so downtime is captured inside observed time.
  • Manual Entry view hides timer controls and timestamp columns so the table can be filled from handwritten studies, spreadsheets, or operator notes.
  • Collapse shows the first and most recent event while keeping the full data available in exports.

Downtime Code Setup

Reason Codes
Define downtime codes here before or during data collection. Use short abbreviations such as JAM, WAIT, QA, or MAT. These codes become selectable in the Event Log and are used to build the Pareto summary.
  • Code abbreviation is the short reason code assigned to each downtime event. Each abbreviation must be unique.
  • Code description explains what the reason code means for anyone reviewing the data later.
  • Include in Summary controls whether a code/tag is included in downtime calculations, the Pareto summary, and the executive report. Raw events and exports still keep the original code.
  • Collapse shows the first and most recent code while retaining all definitions for exports and event dropdowns.
  • Import Code CSV accepts columns like Code #, Code Abbr, Code Desc, and optional Include in Summary.
Paste a downtime-code table copied from Excel or a CSV file. Accepted columns include Code #, Code Abbr, Code Desc, and optional Include in Summary. Tab-separated Excel paste and comma-separated CSV are both supported.
